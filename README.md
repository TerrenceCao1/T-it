- **tit init** - Create an empty T-it repository or reinitialize an existing one
- **tit hash_object** - Creates hashes (which are used for identifiers) for blobs, commits, trees, tags 
- **tit add** - Adds a file to the staging area (index)
- **tit status** - Tells us what files are staged
- **tit write_tree** - Creates the actual tree of files in the staging area, and will become a commit
- **tit commit** - Creates the commit object with tree, pointers to previous commits, a message, and metadata
- **tit log** - Read out the commit history of a repository
